THREE WAYS TO ADD css

  1. INLINE-CSS
  2. INTERNAL-CSS
  3. STYLE.CSS-PAGE

IN inline css we need to change line by line using style attribute

IN INTERNAL-CSS we need to select a selector like div span or than change at once using this code

[{ div,span color: chartreuse; background-color: darkorange; font-size: 20px; text-align: center; text-underline-position: below; }]

IN EXTERNAL-CSS we need to create a .css page and link this page to .html page
